# 更新日志

## 未发布

### 变更

- 主界面重构为深色模块工作台，工程转换与 Keil TI 环境配置通过侧栏作为两个独立功能切换。
- 重新组织工程、开发环境、输出目录、转换状态和构建日志的信息层级，精简重复说明并完善窄窗口布局。
- 整体提高界面字号，改善路径、状态、日志和操作说明的可读性。
- 新增界面设置模块，提供 TI 红、科技蓝、青绿色和紫色四套 MD3 风格全局配色预设，统一切换背景、Surface、边框、文字与强调色，并在本机保存选择。

### 构建

- GitHub Actions 增加 Rust/Cargo 构建缓存，缩短后续 Windows 构建时间。
- Release 产物统一使用 ASCII 文件名，确保自动更新清单能正确关联安装包和签名。

## 0.1.0 - 2026-07-22

首个公开版本。

### 新增

- 支持 CCS 与 Keil 工程双向转换。
- 自动检测 MSPM0 SDK、Pack、CCS、Keil 和 SysConfig。
- 支持转换前后的 CCS/Keil 工程构建验证与实时日志。
- 支持一键配置 Keil TI SysConfig 环境，并在修改前自动备份。
- 提供 Windows x64 安装版和便携版。
- 安装版支持检查、下载和安装新版本。
- 增加“关于”窗口、作者主页和项目仓库入口。
- 增加手动检查更新按钮，更新前展示当前版本对应的更新内容。

### 使用要求

- 工程互转时，源工程和目标环境必须使用相同版本的 SysConfig。
- 当前主要支持普通 NoRTOS DriverLib 工程。
